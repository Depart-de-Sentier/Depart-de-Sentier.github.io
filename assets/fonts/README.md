# Font Files Required

Upload the following font files to this folder for the DdS brand typography:

- **UrbaneRounded-Light.woff2** (preferred, modern browsers)
- **UrbaneRounded-Light.woff** (fallback for older browsers)

The site uses "Urbane Rounded Light" as the primary brand font. Without these files, the site will fall back to system fonts and may look different on production vs. local.
